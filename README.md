# 🌊 Rising Waters - Flood Prediction System

## 📌 Project Overview

Rising Waters is a Machine Learning-based Flood Prediction System developed using Python and Flask. The application predicts whether flood conditions are likely to occur based on weather and rainfall parameters.

The project helps users analyze environmental conditions and receive an instant flood prediction through an easy-to-use web interface.

---

## 🚀 Features

- Predicts flood occurrence using Machine Learning
- User-friendly web interface built with Flask
- Instant prediction results
- Attractive responsive design
- Background image and modern UI
- Uses historical rainfall and weather data

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn
- NumPy
- Pickle

---

## 📊 Input Parameters

The system predicts floods using the following parameters:

- Temperature
- Humidity
- Cloud Cover
- Annual Rainfall
- Jan-Feb Rainfall
- Mar-May Rainfall
- Jun-Sep Rainfall
- Oct-Dec Rainfall
- Average June Rainfall
- Subdivision Rainfall

---

## 🤖 Machine Learning Model

- Algorithm: Random Forest Classifier
- Data Scaling: StandardScaler
- Model Storage: Pickle (.save)

## ▶️ Run Locally

1. Install the required packages

```bash
pip install -r requirements.txt
```

2. Run the application

```bash
python app.py
```

3. Open in your browser

```
http://127.0.0.1:5000
```

## 📷 Output

The application displays either:

- ✅ No Flood Chance
- ⚠️ Flood Chance Detected

based on the weather conditions entered by the user.
